# 🖋️ مُولّد أوامر الخط العربي للذكاء الاصطناعي (Arabic Calligraphy Prompt Generator) (تكوين | Takween)

أداة ويب احترافية ومتقدمة صُممت خصيصاً لهندسة الأوامر (Prompt Engineering) الموجهة لنماذج توليد الصور بالذكاء الاصطناعي (مثل Midjourney و DALL-E 3 وغيرها). تهدف الأداة إلى مساعدة المصممين والفنانين في استخراج **مخطوطات عربية جرافيكية أصيلة ومستقلة**، بعيداً عن القوالب الجاهزة أو الزخارف العشوائية التي تفرضها نماذج الذكاء الاصطناعي عادةً.

## ✨ المميزات الرئيسية

* **📑 قوالب نصية متعددة:** 
  يدعم التطبيق توليد أوامر مخصصة لخمس حالات استخدام مختلفة:
  1. **الأسماء:** توليد أوامر للأسماء المفردة أو المتعددة بأسلوب التوقيع أو المخطوطة المستقلة.
  2. **النص المنثور:** للفقرات وعبارات التكريم، مع تحكم دقيق في التوزيع البصري والمحاذاة.
  3. **الشعر:** معالجة ذكية للأبيات الشعرية، مع دعم نظام الشطرين (الصدر والعجز) وفصل العناوين.
  4. **السطور المؤثرة:** للحكم والاقتباسات، مع إبراز تسلسلي وتذييلات أنيقة.
  5. **التوقيع:** لتصميم أختام أو تواقيع اسمية معتمدة على الهيكلية الخطية.

* **🎛️ تحكم دقيق بالخصائص الفنية:**
  يتيح لك التطبيق ضبط أدق تفاصيل المخطوطة:
  - **المستوى الخطي:** من الكلاسيكي الصارم إلى الحر المعاصر.
  - **الطابع الخطي:** (ثلث، نسخ، ديواني، كوفي، إلخ).
  - **المعالجة اللونية والكثافة البصرية.**
  - **مرونة التوزيع ونمط الأرقام.**

* **🧩 استقلالية العناصر (Smart Element Isolation):**
  هندسة أوامر ذكية تضمن فصل العناوين، النصوص الأساسية، والتذييلات. كل عنصر يحتفظ بمحاذاته وخصائصه (مثال: عنوان محاذٍ لليمين، ونص مضبوط كلياً) دون أي تداخل أو تشويه من الذكاء الاصطناعي.

* **🔤 التشكيل النحوي الإلزامي:**
  توجيهات صارمة تجبر نماذج الذكاء الاصطناعي على تشكيل الكلمات بالحركات النحوية الصحيحة قبل الشروع في التصميم، لضمان أصالة المخطوطة.

* **🛡️ قيود سلبية صارمة (Negative Constraints):**
  نظام قيود ذكي يمنع الذكاء الاصطناعي من "الهلوسة" وإضافة إطارات، خلفيات مزخرفة، أو تحويل النص إلى تصميم بطاقة كاملة، لضمان خروج النص كأصل جرافيكي (Asset) قابل للتفريغ والاستخدام.

* **🌐 مخرجات ثنائية اللغة:**
  توليد الأوامر باللغتين العربية والإنجليزية لتناسب مختلف محركات الذكاء الاصطناعي.

## 🚀 طريقة الاستخدام

1. قم بتحميل ملف `index.html` وافتحه في أي متصفح ويب حديث (لا يحتاج إلى خادم محلي).
2. اختر التبويب المناسب لنوع النص الذي تريد تصميمه (اسم، شعر، نثر...).
3. أدخل النصوص في الحقول المخصصة (يمكنك استخدام `||` للفصل بين الأسطر).
4. قم بضبط الإعدادات الفنية من القائمة الجانبية لتناسب رؤيتك.
5. انسخ البرومبت (العربي أو الإنجليزي) والصقه في محرك توليد الصور المفضل لديك.

## 🛠️ التقنيات المستخدمة
تم بناء هذه الأداة كملف تنفيذي واحد (Single-page Application) باستخدام:
- **React.js** (واجهة المستخدم)
- **Tailwind CSS** (التصميم والتنسيق)
- **Framer Motion** (الحركات والتفاعلات السلسة)
- **Lucide Icons** (الأيقونات الجمالية)
- **Vite** (بيئة التطوير والبناء)
